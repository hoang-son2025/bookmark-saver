# Bookmark Saver

シンプルなブックマーク保存アプリです。ブラウザの localStorage を使って名前と URL を保存します。

## 使い方
1. プロジェクトの [index.html](index.html) をブラウザで開きます（ライブサーバー拡張機能を使うと便利です）。
2. 「Bookmark Name」に名前を入力し、「Bookmark URL」に URL（http:// または https:// で始まるもの）を入力します。
3. 「Add Bookmark」ボタンを押すとリストに追加され、ブラウザの localStorage に保存されます。
4. 保存したブックマークはリストの「Remove」ボタンで削除できます。
